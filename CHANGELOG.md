# Changelog

## 0.2.0 - 2026-06-26

- Added Ubuntu Touch Content Hub support.
- Shared text from other apps can now be imported as a new task.
- Tasks can now be shared to other apps, using task title and notes where available.

## 0.1.0 - 2026-06-21

- Very early first public release.
- Connects to Nextcloud Tasks through Ubuntu Touch Online Accounts.
- Supports basic task and list management, local cache, autosave, and manual sync while the app is open.
- Includes light/dark theme support and early translation coverage.
- Some rough edges are expected; feedback and bug reports are welcome.

## 0.0.1 - 2026-06-14

- Initial Ubuntu Touch scaffold for NextTasks.
- Added Clickable/CMake project structure.
- Added QML placeholder UI and C++ launcher.
- Added minimal AppArmor/accounts/networking setup.
- Added themed icon and OpenStore banner draft.
- Added gettext translation structure.
